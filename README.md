# Informe-TRM
Problema: La volatilidad constante del dólar genera incertidumbre financiera. Sin una visualización histórica, es complejo analizar cómo las fluctuaciones impactan los costos operativos y la planeación presupuestal a largo plazo.

Solución: Creación de un monitor de TRM que conecta datos históricos para visualizar picos, valles y tendencias de la moneda. Esta herramienta permite a los analistas financieros anticipar comportamientos del mercado y ajustar las estrategias de cobertura cambiaria
